Financial Ratios:

  • Current Ratio = {{currentRatio}}

    Current assets / Current liabilities = {{currentAssets}} / {{currentLiabilities}}

  • Quick Ratio = {{quickRatio}}

    (Current assets-Inventory) / Current Liabilities = ({{currentAssets}} - {{inventory}}) / {{currentLiabilities}}

  • Inventory to Net Working Capital = {{inventoryToNetWorkingCapital}}

    Inventory /(Current assets - Current Liabilities) = {{inventory}} / ({{currentAssets}} - {{currentLiabilities}})

  • Debit to Asset Ratio = {{debtToAssetRatio}}

    Total debt / Total assets = {{totalLiabilities}} / {{totalAssets}}

  • Debt to Equity Ratio = {{debtToEquityRatio}}

    Total debt / Total stockholders' equity = {{totalLiabilities}} / {{stakeHolderEquity}}

  • Long Term Debt to Equity Ratio = {{longTermDebtToEquityRatio}}

    Long-term debt / Total shareholders' equity = {{longTermLiabilities}} / {{stakeHolderEquity}}
